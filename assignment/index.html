OBJECT

Object is a collection of properties and these properties are basically key-value pairs. Properties defines the characteristic of an object. These objects are mutable, in another word they can be changed, deleted, and addedeven after object has been created!

Accessing Object

  • Using Dot Notation
  • Using Square ([]) Bracket Notation
  • Using Object.defineProperty() Method

Object Methods

  • Objects can also have methods.
  • Methods are actions that can performed on objects.
  • Methods are stored in properties as function definitions.
  • A method is a function stored as a property.

Nested Arrays and Objects

Values in objects can be arrays and values in arrays can be objects.

Adding New Properties

New properties can be added to an existing object by giving it a value.
